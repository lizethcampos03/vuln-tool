# CVE Dataset Notes

This folder stores CVE-related vulnerability records or curated CVE examples used to support the RAG layer.

CVEs describe real-world vulnerabilities in specific products, libraries, or systems. Because CVE data can be very large and product-specific, the current prototype uses curated examples and leaves space for later expansion.

For the current Phase 2 calibration, CWE entries are prioritized because they map more directly to vulnerability weakness types in source code.

Future work:
- Add CVE records relevant to tested libraries and frameworks
- Link CVEs to related CWE categories
- Expand retrieval beyond curated examples
